# AMR 追踪位置显示 - 操作步骤

## 问题

地图上 AMR 的位置和朝向没有显示

## 解决方法

### 步骤 1：验证配置

确保 `topics.txt` 包含 `/tracked_pose`：

```bash
cat topics.txt
```

应该看到：
```
/tracked_pose
/battery_state
/map
```

### 步骤 2：启动应用

```bash
python main.py
```

### 步骤 3：打开地图查看器

点击主窗口的 "📍 显示实时地图" 按钮

### 步骤 4：观察地图

应该看到：
- ✓ 绿色点 + 叉：坐标原点 [0, 0]
- ✓ 蓝色箭头：AMR 当前位置和朝向
- ✓ 状态栏显示：位置和朝向信息

## 运行测试

验证功能是否正常：

```bash
python test_integration.py
```

输出应该显示：
```
✓ 所有集成测试通过！
```
