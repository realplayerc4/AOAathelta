🚀 最快的开始方式 (5 分钟):

1. 启动应用:
   $ python main.py

2. 在 UI 中:
   - 点击 "📡 AOA 数据" 标签页
   - 选择串口 (默认 /dev/ttyUSB0)
   - 点击 "🔌 连接"
   - 查看实时数据

3. 更多信息:
   - 查看 AOA_README.md

• Python 3.8+
• PyQt6
• pyserial (用于串口通信)
• 硬件: AOA 设备 (或模拟)

验证安装:
